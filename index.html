Tauri Masterclass: Build Native Desktop Apps

Welcome to the ultimate beginner's guide to Tauri. Learn how to build blazing fast, ultra-lightweight desktop applications using standard web technologies (HTML, CSS, JavaScript).

Table of Contents

  • 1. Prerequisites & Installation

1. Prerequisites & Installation

Before building a native app, your computer needs the right tools installed. Do this once, and you are set forever.

  1. Node.js: Download and install from nodejs.org.
  2. Rust: Download and install rustup from rustup.rs.
  3. Microsoft C++ Build Tools (Crucial for Windows):
    • Download the Visual Studio Installer.
    • Select "Desktop development with C++".
    • Pro-tip to save space: Uncheck everything on the right side EXCEPT MSVC v143... and Windows 11 SDK. This reduces the download from 7GB to roughly 2GB!
Important: After installing these tools, you MUST restart your command prompt or terminal!

2. Creating Your First Project

Open your terminal and run the command to scaffold a new Tauri app. We will use Vanilla JavaScript for simplicity.

npm create tauri-app@latest my-app

Follow the prompts: Choose npm -> Vanilla -> JavaScript. Once created, navigate into the folder and install dependencies:

cd my-app
npm install

3. Running in Development Mode

To see your app live and test the sample template, run the development server. The first time you run this, Rust will take a few minutes to download and compile core libraries. Future runs will be instant.

npm run tauri dev

A native desktop window will appear showing the default Tauri template. Any changes you make to the src/index.html file will instantly update the app!

4. Building the Final Release

When you are ready to share your app with the world, it is time to build the highly optimized, tiny executable file.

npm run tauri build

Once finished, you can find your final applications here:

  • The .exe file: src-tauri/target/release/my-app.exe (Usually under 10MB!)
  • The .msi Installer: src-tauri/target/release/bundle/msi/ (A professional setup wizard for your users).

5. Creating a Portable App

Want an app that runs directly from a USB drive without installation?

  1. Create an empty folder on your Desktop.
  2. Go to src-tauri/target/release/
  3. Copy my-app.exe and WebView2Loader.dll into your new folder.
  4. That's it! You can ZIP this folder and run the app anywhere.
